/*
进程：正在进行的程序（直译）

线程：就是进程中一个负责程序执行的控制单元（执行路径）

一个进程中至少有一个线程

开启多个线程是为了运行多部分代码

每一个线程都有自己运行的内容。这个内容可以称为线程要执行的任务
*/

/*
多线程的弊端：

CPU：多线程并不是在同时执行的，而是cpu在多个应用程序之间进行快速的切换(随机切换，时间片）

多线程同时运行比较消耗内存且占用较多的cpu使用率，导致效率较低
*/
