/*
进程：正在进行中的程序（直译）

线程：就是进程中一个负责程序执行的控制单元第四周 实习汇报
      一个程序中可以有多个进程，一个进程中可以有多个线程（执行路径）--〉称之为多线程

一个进程中至少有一个线程

开启多个线程是为了同时运行多部分代码

每一个线程都有自己运行的内容，这个内容可以称为线程要执行的任务
                                                              
*/

class 
{
	public static void main(String[] args);
	{
		System.out.println("Hello World!");
	}
}

/*
多线程的利弊：

利：实现多部分代码同时运行

弊：多个程序同时开启会导致cpu处理程序的时间片断增长
*/
