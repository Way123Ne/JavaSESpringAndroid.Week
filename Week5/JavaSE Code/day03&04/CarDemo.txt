/*
用java语言对现实生活中的事物进行描述
通过类的形式来体现的。

怎么描述事物？
对于事物描述通常只关注两个方面
一个是属性，一个是行为

只要明确该事物的属性个性为并定义在类中即可

对象：就是该类事物实实在在的个体

类与对象之间的关系？
类：事物的描述
对象：就是该类事物实实在在的个体。在java 中通过new来创建的
*/

/*
描述小汽车：

分析属性和行为
1.属性
   轮胎数
   颜色
2.行为
   运行


定义类其实就是定义类中的成员
成员：成员变量<-->属性；成员函数<-->行为

成员变量和局部变量的区别：

1.
成员变量定义在类中，整个类中都可以访问
局部变量定义在函数、语句、局部代码块中，只在所属的区域有效

2.
成员变量存在堆内存的对象中
局部变量存在栈内存的方法中

3.
成员变量随着对象的创建而存在，随着对象的消失而消失
局部变量随着所属区域的执行而存在，随着所属区域的消失而释放

4.
成员变量都有默认初始化值
局部变量没有默认初始化值
   */

class CarDemo2
{
	public static void main(String[] args)
{
		//在计算机中创建一个car 的实例。通过new 来完成
	Car c=new Car();  //c就是一个类类型的引用变量，指向了该类的引用对象（重要：数组类型的引用变量）
	c.num=4;
	c.color="blue";

	c.run(); //要使用对象中的内容，可以通过对象成员的形式来调用

}
}


class Car
{
	int num;  // int num=4  显示初始化
	String color;

	void run()
	{
		//int num=8;//成员变量和局部变量同名
		System.out.println(num+"..."+color);
	}
}
